Monthly Revenue and Order Volume Analysis

## Project Overview
This project analyzes monthly revenue and order volume using SQL on the online sales dataset.

## Dataset
Table Name: `sales_csv`

Columns Used:
- Order ID
- Order Date
- Sales

## Objectives
- Analyze monthly revenue.
- Analyze monthly order volume.
- Group data by year and month.
- Sort results chronologically.

**##Functions Used**
EXTRACT()
SUM()
COUNT(DISTINCT)
GROUP BY
ORDER BY

**Output**
The query generates:
Monthly Revenue
Monthly Order Volume
Year-wise and Month-wise sales summary

**Tools Used**
MySQL Workbench
SQL

**Conclusion**
The analysis provides a month-wise summary of sales performance and order volume, helping identify trends and business growth patterns over time.
